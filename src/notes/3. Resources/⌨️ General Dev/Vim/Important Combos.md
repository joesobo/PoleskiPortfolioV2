## Key Combos
- `hjkl` - Navigate
- `y` - Copy
	- `yy` - copy line without going into visual mode
- `p` - Paste
- `/{term}` - Search

## Cool Key Combos
- `Ctrl + e` - scroll down
- `Ctrl + y` - scroll up
- `c + i + w` - change word from any point within word
- `d + i + w` - delete word from any point within word
- `c + i + p` - change inside paragraph
- `c + i + '` - change inside ''
